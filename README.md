# 音声解析アプリ

## 概要
音楽の中身の分析を目的として作成した音声解析アプリ

## 使用技術
- 言語：HTML/CSS,JavaScript,Python
- ライブラリ：wavesurfer(JavaScript)
- FW：FastAPI(Python)

## 工夫した点
- 多言語間でのファイルのやり取りを実装した。
- PythonでのAPI実装時に、routerを使用して拡張しやすく設計したこと
- APIの処理待ち時にアニメーションを追加して、UI・UXを少しだけ意識したこと

## 今後の改善点
- バイナリデータの扱いが難しく、必要に応じ、適切なライブラリを使用する必要があると感じた
- 現状wav音声でしか解析できないため、ライブラリを使用してすべての音声ファイルを解析できるようにする
- 解析項目があまり実用的でない。
- 処理速度の高速化
